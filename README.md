# Сервис для напоминаний в телеграмм боте
***
## Описание

Данное приложение прелставляет собой сервис напоминаний с помощью телеграмм бота. Пользователи могут создавать напоминания, получать уведомления в заданное время. Просматривать свои напоминания и удалять.

Основные функции:

- Пользовательский флоу:

  1. Пользователь создаёт напоминание (текст + дата/время).
  2. В указанное время приходит уведомление.
  3. Пользователь может повторить напоминание (через 5 минут, через 1 час, через 1 день)
  4. Пользователь может просматривать и редактировать список своих напоминаний.

- Флоу админимтратора:

  1. Админ может просматривать список всех напоминаний (например, через админку или бота).
  2. Возможность блокировать пользователей с указанием причины.

## Установка

### 0. Требования
- docker-desktop
### 1. Настройка
1.1. Клонирование репозитория
```
git clone https://github.com/JellyBean-bit/reminder-service.git
```
1.2. В корне проекта добавьте файл .env, в котором необходимо рописать переменные окружения
```
BOT_TOKEN=BOT_TOKEN

REDIS_URL=redis://redis:6379/0
REDIS_HOST=redis
REDIS_PORT=6379

DATABASE_URL=sqlite:///reminders.db

ADMINS=список_id_админов
```
***
### Запуск проекта

1. Для запуска проекта неолбходимо в корне проекта пропсиать:
```
docker-compose up --build
```
2. После запуска проекта перейдите в телеграмм бот.
